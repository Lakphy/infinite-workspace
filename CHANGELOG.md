# Changelog

All notable changes to the "Infinite Workspace" extension will be documented in this file.

## [0.0.1] - 2026-03-31

### Added

- Initial release
- Infinite canvas with pan and zoom
- Draggable terminal windows with real PTY backend
- Built-in browser windows with navigation toolbar
- File explorer windows with workspace directory browsing
- Smart snap guides for window alignment
- Context menu for creating windows at cursor position
- Favorites system for terminal commands, URLs, and file paths
- Settings dialog (grid toggle, snap configuration)
- State persistence across sessions
- VS Code Web (GitHub Codespaces) support with graceful degradation
- Dark theme matching VS Code's native aesthetic
